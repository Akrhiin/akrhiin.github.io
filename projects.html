Projects

Health Data Correlation Calculator

The program takes user input of three (x,y) pairs of numbers as (temperature(F), respiratory rate(BPM)) and uses a formula to calculate the correlation coefficient and the strength of association, designating the points as having strong, moderated, or weak relation. The program was created as part of a C++ course in college, and is still functional as standalone code.

Massachusetts Vaccination Notifications

A Python script that monitors the Massachusetts vaccine finder website for available appointments at specific locations and sends SMS notifications when slots become available. Currently configured for the Marshfield vaccination site. Massachusetts vaccine system has since been updated, so this project is nonfunctional.

Webcam-based Facial Recognition

A Python application that performs real-time face detection through a webcam using OpenCV and Haar Cascade Classifiers. Supports front-facing and profile face detection with multiple cascade classifiers for improved accuracy.

Day/Night Image Classifier

A machine learning model build in Python with Scikit-learn and OpenCV. It classifies images as having been taken during the day or night, providing probability scores for each classification. The project was created for an artificial intelligence course, and is functional.
